what is JavaScript

JavaScript (js) is a light-weight object-oriented programming language which is used by several websites for scripting the webpages. It is an interpreted, full-fledged programming language that enables dynamic interactivity on websites when applied to an HTML document. It was introduced in the year 1995 for adding programs to the webpages in the browser. Since then, it has been adopted by all other graphical web browsers. With JavaScript, users can build modern web applications to interact directly without reloading the page every time. The traditional website uses js to provide several forms of interactivity and simplicity.

In JavaScript we used 3 Types of variables

  1. Var:The most common variable. Can be reassigned but only accessed within a function. Variables defined with var move to the top when code is executed.
  2. let: let variable can be reassigned but not re-declared.
  3. Const: Cannot be reassigned and not accessible before they appear within the code.
